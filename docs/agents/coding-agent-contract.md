# Coding Agent Contract

1. Read the issue/user request and inspect relevant files before editing.
2. State a concise implementation plan after inspection.
3. Keep the implementation within scope.
4. Make atomic Conventional Commits.
5. Run `bun run check` and `bun run typecheck` unless a narrower verified gate is justified.
6. Open a PR with summary, test plan, stack notes, and issue closure/reference.
